# AIRSM(A-IOT Sorting Machine) 디바이스 프론트엔드



## dir 구조

아래 대표 화면 컴포넌트는 src 안에 포함하였고, 그 하위 컴포넌트들은 src/components 안에 포함하였다.



## 외부 라이브러리

- React-Route
- Material-UI
- MQTT



## 대표 화면 구성

사용자는 다음 순서대로 행동하게 된다.

1. 휴대폰 입력 화면
2. 가이드라인 화면
3. 위반 가이드라인 화면
4. 배출구 안내 화면
5. 적립 안내 화면





## 화면별 사용 기술

React-Route

function component에서의 statehook을 이용한 상태 변경

## 사용법
`yarn` 명령어를 통해 패키지 설치
